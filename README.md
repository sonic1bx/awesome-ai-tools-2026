## Code Assistants

AI-powered tools to help you write better code faster.

- **[GitHub Copilot](https://github.com/features/copilot)** - AI pair programmer that suggests code and entire functions in real-time. Supports 70+ languages.
- **[Cursor](https://cursor.sh/)** - AI-first code editor built on VS Code. Chat with your codebase, generate code, and fix bugs.
- **[Codeium](https://codeium.com/)** - Free AI code completion tool. Fast, accurate, and supports 70+ languages.
- **[Tabnine](https://www.tabnine.com/)** - AI code completion that learns from your code. Works offline for privacy.
- **[Amazon CodeWhisperer](https://aws.amazon.com/codewhisperer/)** - Free AI coding companion from AWS. Includes security scanning.
- **[Replit AI](https://replit.com/ai)** - AI assistant built into Replit. Generate, explain, and debug code.
- **[Sourcegraph Cody](https://sourcegraph.com/cody)** - AI coding assistant that understands your entire codebase.

## Image Generation

Create stunning images with AI.

- **[Midjourney](https://midjourney.com/)** - High-quality AI art generation via Discord. Best for artistic and creative images.
- **[DALL-E 3](https://openai.com/dall-e-3)** - OpenAI's image generator. Excellent at following complex prompts.
- **[Stable Diffusion](https://stability.ai/)** - Open-source image generation. Run locally or use online platforms.
- **[Adobe Firefly](https://www.adobe.com/products/firefly.html)** - AI image generation integrated into Adobe products.
- **[Leonardo.ai](https://leonardo.ai/)** - AI art generator with fine-tuned models for different styles.
- **[Ideogram](https://ideogram.ai/)** - AI image generator that's great at text rendering.

## Writing & Content

AI tools for writing, editing, and content creation.

- **[ChatGPT](https://chat.openai.com/)** - Conversational AI for writing, brainstorming, and problem-solving.
- **[Claude](https://claude.ai/)** - Anthropic's AI assistant. Great for long-form content and analysis.
- **[Gemini](https://gemini.google.com/)** - Google's multimodal AI. Integrates with Google Workspace.
- **[Jasper](https://www.jasper.ai/)** - AI writing assistant for marketing and business content.
- **[Copy.ai](https://www.copy.ai/)** - AI copywriting tool for marketing, sales, and social media.
- **[Grammarly](https://www.grammarly.com/)** - AI-powered writing assistant for grammar, tone, and clarity.
- **[Notion AI](https://www.notion.so/product/ai)** - AI assistant built into Notion for writing and organizing.

## Video & Audio

AI tools for video and audio creation.

- **[Runway](https://runwayml.com/)** - AI video editing and generation. Create videos from text prompts.
- **[Descript](https://www.descript.com/)** - AI-powered video and podcast editing. Edit audio by editing text.
- **[ElevenLabs](https://elevenlabs.io/)** - AI voice generation and cloning. Natural-sounding text-to-speech.
- **[Synthesia](https://www.synthesia.io/)** - Create AI videos with virtual presenters.
- **[Pictory](https://pictory.ai/)** - Turn long-form content into short videos automatically.
- **[Murf AI](https://murf.ai/)** - AI voice generator for voiceovers and narration.

## Data & Analytics

AI tools for data analysis and insights.

- **[Julius AI](https://julius.ai/)** - AI data analyst. Upload data and ask questions in natural language.
- **[Tableau Pulse](https://www.tableau.com/products/tableau-pulse)** - AI-powered analytics insights from Salesforce.
- **[Microsoft Copilot](https://www.microsoft.com/en-us/microsoft-365/copilot)** - AI assistant in Excel, PowerPoint, and other Office apps.
- **[DataRobot](https://www.datarobot.com/)** - Automated machine learning platform.
- **[H2O.ai](https://h2o.ai/)** - Open-source AI and ML platform for data science.

AI-powered development and DevOps tools.
