# Contributing to Awesome AI Tools 2026

Thank you for your interest in contributing! This document provides guidelines for contributing to this awesome list.

## How to Contribute

1. **Fork the repository**
2. **Create a new branch** (`git checkout -b add-new-tool`)
3. **Make your changes**
4. **Commit your changes** (`git commit -m 'Add NewTool to Code Assistants'`)
5. **Push to your branch** (`git push origin add-new-tool`)
6. **Open a Pull Request**

## Guidelines

### Adding a New Tool

- **Quality over quantity**: Only add tools you've personally used or that are widely recognized
- **Keep descriptions concise**: 1-2 sentences maximum
- **Include the official link**: Link to the tool's official website
- **Categorize correctly**: Place the tool in the most appropriate category
- **Alphabetical order**: Maintain alphabetical order within each category (optional but nice)

### Format

```markdown
- **[Tool Name](https://official-link.com/)** - Brief description of what it does and why it's useful.
```
